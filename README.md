#Patrones de navegación

##1. navegación

  Relación padre-hijo  
  Uso: mediante botones o enlaces  
  ```html
  <ons-navigator> o <ons-toolbar>
  ```
  
##2. menú desplazable
  Se usa cuando necesitamos muchas páginas en el mismo nivel 
  Uso: mediante desplazamiento o botón en barra superior
  ```html
  <ons-sliding-menu>
  ```
